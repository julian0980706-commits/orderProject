git config --list 查看設定
git config --global user.name "(userName)"	
git config --global user.name "(e-mail)"	
git init
git status
git add filename
git add . 將全部工作區檔案移去暫存區(還沒存) 除了ignore裡的

git add *js 副檔名是js的
git add secret*js開頭是secret副檔名是js的
git add *login* 檔名有login的
git add css/*加入 css 資料夾下所有東西
git add img/*.png加入 img 資料夾內所有 PNG 圖
!!!雖然 * 很方便，但它有一個致命缺點：它會忽略.gitignore

git commit -m"" 將暫存區資料存起來

git log查詢歷史紀錄
git log --oneline查詢歷史紀錄(單行顯示 資訊較不複雜)
!!進入git log 或其他模式時，會進入閱讀模式(上下鍵閱讀)
按q離開
顯示由上到下，從新到舊

HEAD是現在位置
master是指最新
git checkout 版本號 時空跳轉到版本
git checkout master(回到最新)
git checkout 版本號 my.txt (將檔案改成那時候的狀態，且自動幫忙add)!!不改變HEAD改變了工作區和暫存
git checkout -- my.txt (回到最近一次存檔(add or commit)的位置 )
git checkout -- .(回到最近一次存所有檔(add or commit)的位置)
git reset --hard 版本號 (等於時空跳轉而且把跳轉到的時空之後的東西都刪掉)
